# Phishing Simulation

## Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://localhost:5000/

4. Interact with the safe and phishing pages.
   Logs will be saved in `data/user_logs.csv`.

5. This is an app to generate a synthetic data(user behavior metrics) to train my ML module for my reseach project. 
   Every interaction/action made on the app is recorded soley for academic purposes.
